# Notificación por renombramiento del repositorio

He renombrado el repositorio en GitHub de `investment-manager-firebase` a `home-flow`.

Si tienes una copia local del repo, por favor actualiza el remoto `origin` ejecutando:

```bash
# SSH
git remote set-url origin git@github.com:albert-carrasquel/home-flow.git
# o HTTPS
git remote set-url origin https://github.com/albert-carrasquel/home-flow.git

git fetch --all
```

Revisa también integraciones externas (Netlify/Vercel, webhooks, badges en README) y reporta cualquier fallo.

Gracias.
